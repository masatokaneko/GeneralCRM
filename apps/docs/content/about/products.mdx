# Welcome to My Project Documentation

This is a comprehensive documentation site built with [Nextra](https://nextra.site) and Next.js.

## Overview

This monorepo contains multiple applications and services:

- **Web Application**: Next.js 15 application with React 19
- **MCP Server**: Model Context Protocol server for AI development tools
- **Documentation**: This Nextra-powered documentation site

## Quick Start

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development servers
npm run dev
```

### Available Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start all development servers |
| `npm run build` | Build all packages |
| `npm run lint` | Run linting across all packages |
| `npm run test` | Run tests for web application |
